# SeaPilotApp Privacy Policy (GitHub Pages)

This repository hosts the public Privacy Policy required by Google Play.

## Files
- `docs/index.html` – auto-redirects by browser language
- `docs/privacy-policy-en.html` – English
- `docs/privacy-policy-uk.html` – Ukrainian

## Enable GitHub Pages
1. Push this repo to GitHub.
2. Open **Settings → Pages**.
3. **Build and deployment → Source**: select **Deploy from a branch**.
4. **Branch**: `main` and folder `/docs`.
5. Save. GitHub will show the public URL.

## Google Play link
Use the published URL, for example:
`https://<your-username>.github.io/<repo-name>/`
or direct:
`https://<your-username>.github.io/<repo-name>/privacy-policy-en.html`
